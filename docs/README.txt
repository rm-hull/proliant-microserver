1. English

Welcome to the Documentation CD

This CD allows you to view, search, and print your user documentation.
     
System Requirements:
- 486 processor or faster
- Windows 2000 or higher
- 128 MB RAM or greater 
- VGA monitor, 800 x 600 resolution or higher (16 bit color recommended)
- 4x CD-ROM drive
- Requires Windows Explorer 5, Netscape 6 or higher

Refer to the documentation set, shipped with this server, for additional details on this and other products.


To install the utility to access the documentation:

1.	Insert the CD into your CD-ROM drive.

2.	Your operating system detects Autorun to start the CD.

If you have disabled Autorun on your system:

1.	Insert the CD into your CD-ROM drive.

2.	Select Start, Run.

3. 	Type <drive>:\START.HTML, where <drive>is the letter on your CD-ROM drive.

2. Deutsch

Willkommen auf der Compaq Server Documentation CD

Mit dieser CD können Sie die Benutzerdokumentation anzeigen, suchen und drucken.

Systemvoraussetzungen:

- 486-Prozessor oder schneller
- Windows 2000 oder höher
- 16 MB RAM oder mehr 
- VGA-Monitor, Auflösung 640 x 480 oder höher (empfohlen: 16-Bit-Farbtiefe)
- 4fach CD-ROM-Laufwerk

Weitere Einzelheiten über diesen Server und andere Produkte entnehmen Sie der Dokumentation, dien zum Lieferumfang des Servers gehört.

So installieren Sie das Dienstprogramm zum Zugriff auf die Dokumentation:

1.	Legen Sie die CD in das CD-ROM-Laufwerk ein.

2.	Das Betriebssystem erkennt die Option Autorun zum Starten der CD.

Falls die Option Autorun auf Ihrem System deaktiviert ist:

1.	Legen Sie die CD in das CD-ROM-Laufwerk ein.

2.	Wählen Sie Start, Ausführen.

3. 	Geben Sie <Laufwerk>:\START.HTML ein; dabei ist <Laufwerk> der Laufwerksbuchstabe Ihres CD-ROM-Laufwerks.
