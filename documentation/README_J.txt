ドキュメンテーションCDについて

このCDを使用して、ユーザ マニュアルを表示、検索、および印刷することができます。
     
システム要件
- 486以上のプロセッサ
- Windows 2000以上
- 128MB以上のRAM
- 解像度800×600以上のVGAモニタ（16ビット カラーを推奨） 
- 4倍速以上のCD-ROMドライブ
- Windows Internet Explorer 5またはNetscape 6以上が必要

この製品およびその他の製品について詳しくは、このサーバに付属のマニュアルを参照してください。


マニュアルを表示するためのユーティリティをインストールするには、以下の手順に従ってください。


1. CDをCD-ROMドライブに挿入します。

2. オペレーティング システムがAutorunを検出して、CDを起動します。

ご使用のシステムでAutorunが無効になっている場合は、次の手順に従ってください。

1. CDをCD-ROMドライブに挿入します。

2. [スタート]メニューから[ファイル名を指定して実行]を選択します。

3. <ドライブ文字>:\START.HTMLと入力します。ここで<ドライブ文字>は、CD-ROMドライブのドライブ文字です。
